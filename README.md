## Get started 🏁

Clone the project and go to the project directory

```bash
git clone git@github.com:teziovsky/react-vite-ts-template.git && cd react-vite-ts-template
```

Install dependencies

```bash
npm install
```

Start the application!

```bash
npm run dev
```
